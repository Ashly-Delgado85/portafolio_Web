# portafolio_Web
Investigar el trabajo soñado del Ingeniero(a) en Sistemas a partir del análisis de ofertas laborales reales y, con base en dicha investigación, diseñar y desarrollar un sitio web tipo portafolio/CV profesional o página de servicios, utilizando HTML5 y CSS3, aplicando principios de diseño web, buenas prácticas de desarrollo y diseño responsivo

Responsabilidades principales:
Brindar soporte y supervisión CAD/CAFM como Space Data Administrator.
Gestionar el procesamiento, documentación y seguimiento de actividades relacionadas con factibilidad, estándares, diseño exterior y señalización.
Interactuar con clientes para soluciones de diseño y gestionar aprobaciones de proyectos/iniciativas.

Comunicar el estado y justificación de estándares a PMs, TMs y otros stakeholders.

1) Patrones comunes que aporta esta oferta:
Fuerte enfoque en CAD/CAFM, AutoCAD y gestión de procesos.
Importancia de documentación, trazabilidad y auditoría de procesos.
Trabajo con clientes, stakeholders y proveedores.
Mezcla de competencias técnicas + operativas + comunicación.
Uso de SQL/MSQL solo como apoyo, no como núcleo del rol.

2) Perfil profesional objetivo:
Space Data Administrator / Coordinación de datos espaciales y CAD/CAFM, con enfoque en:

diseño, estándares, documentación, seguimiento de proyectos, coordinación con clientes.

3) Justificación del enfoque del sitio web:
Aunque la oferta #5 se llama “Space Data Administrator”, su enfoque es principalmente CAD/CAFM, AutoCAD, documentación y coordinación de diseño**, por lo que no corresponde a un DBA tradicional de bases de datos. Por ello, se usa como referencia comparativa para justificar que el sitio web se orienta a una postulación ficticia de DBA, alineada con las otras ofertas analizadas (#1–#4), que sí están enfocadas en administración de bases de datos, SQL, seguridad, monitoreo y recuperación.

4)  diferenciadores clave:

Diferenciador 1: Capacidad de analizar ofertas y distinguir perfiles similares en nombre pero distintos en funciones.
Diferenciador 2: Enfoque profesional claro hacia DBA de bases de datos, evitando una propuesta de portafolio ambigua.
